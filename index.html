PIS Website

It provides an intuitive and user-friendly graphical interface for seamlessly updating and monitoring carpark information in real time. With a range of powerful features, including vacancy alerts, file uploading, carpark mapping, and user management, the PIS website offers an efficient solution for streamlined carpark information management.

My Role

  • Front-end development using React and various libraries like MUI and TinyMCE to build 5+ reusable UI components.
  • Managed back-end operations using Node.js with Express.js to handle JSON or XML REST APIs, facilitating communication between our website, Linux server, and various third-party data sources.
  • Optimized data caching with Linux bash scripts and crontab, ensuring efficient storage and retrieval in MySQL.
  • Designed efficient queries to extract data from large MySQL tables, improving data processing speed and reliability.
  • Using Nginx and Apache for professional web hosting, reverse proxying, and load balancing to ensure optimal performance and reliability.

The app integrates push notification alerts and utilizes SMS login for user authentication. Its primary purpose is to streamline the process of managing parking for carpark workers.

My Role

  • Created the prototype of the PIS app using .NET MAUI, ensuring seamless cross-platform functionality.
  • Integrated the app's back-end with the existing Linux server to share data with the Parking Information System Website.
  • Deployed the app on Android and iOS TestFlight, integrating Firebase Crashlytics for enhanced crash detection and reporting.
  • Implemented Firebase Cloud Messaging for push notifications, improving real-time user engagement.
  • Upgraded the Parking Information System Application from Xamarin to .NET MAUI, showcasing adaptability and a quick learning curve.
