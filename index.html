Serverless Application

The Serverless Application is a web-based, cross-platform solution built using Flutter for the frontend and Firebase for the backend. It allows users to easily create, view, and manage events in real-time. The app features a powerful filtering system that enables users to search for events by type, such as "Conference," "Workshop," or "Webinar," as well as by event date. Seamlessly integrated with Firestore for real-time data storage and Cloud Functions for backend logic, the app provides a smooth and intuitive user experience. Explore upcoming events, edit existing ones, or create new ones with just a few clicks.

My Role

  • Built the full-stack application using Flutter for the frontend and Firebase (Firestore, Cloud Functions) for the backend, ensuring real-time event management.
  • Integrated Firestore for real-time data storage and retrieval, and implemented Firebase Cloud Functions for serverless backend operations.
  • Developed a filtering feature to allow users to filter events by type improving user experience and event discovery.
  • Managed state using Provider, ensuring smooth and responsive state management across the app.
  • Implemented robust error handling for network requests and form submissions, ensuring a smooth user experience even under failure conditions.
