PIS APP

The Parking Information System (PIS) is a multi-platform app available on both Android and iOS. It serves as a tool for carpark workers to manually update and monitor parking vacancy in real-time. The app integrates push notification alerts and utilizes SMS login for user authentication. Its primary purpose is to streamline the process of managing parking for carpark workers.

My Role

I developed the Parking Information System (PIS) app using .NET MAUI, adhering to client specifications. The app is available on Android and iOS platforms and integrates Firebase Crashlytics for robust crash reporting. Firebase Cloud Messaging is utilized to deliver push notifications to both platforms. To enable efficient data access, I implemented APIs for seamless communication with the server's database. Also I implemented a SMS-based login system to ensuring a secure authentication process.
